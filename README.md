Essence Loise & Pontes

1. Contexto

O projeto consiste em uma aplicação web de restaurante do tipo landing page, com foco na apresentação do cardápio, destaques de pratos e realização de reservas online. A interface foi pensada para ser responsiva e acessível.

2. Público-alvo

Pessoas entre 18 e 60 anos que desejam visualizar o cardápio e fazer reservas de forma rápida. O principal dispositivo utilizado é o smartphone.

3. Dor principal

Muitos clientes têm dificuldade em acessar o cardápio atualizado e realizar reservas sem precisar entrar em contato direto com o restaurante.

4. Critério de sucesso

O usuário consegue visualizar o cardápio, navegar entre categorias e realizar uma reserva em menos de 2 minutos.
